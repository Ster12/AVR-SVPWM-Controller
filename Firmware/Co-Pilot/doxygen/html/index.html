This program is implemented on ATmega16/32 running at 16 MHZ CPU clock; it basically processes and shows current inverter status like Voltage, Frequency and the speed of the motor and sets the main controller output by communicating via parallel bus using 8-bit data line, one ACK line and one READY line; It also interfaces 16x2 parallel interface character LCD as an output.

Pin Assignments

  • PA0: Speed Potentiometer signal
  • PA1-3: Phases current signal
  • PB0-3: LCD 4-bit Data signal
  • PB5: LCD EN signal
  • PB6: LCD RS signal
  • PB7: Direction Data signal
  • PC0-7: Parallel bus 8-bit Data signal
  • PD0-1: Direction Switch signal
  • PD3: Current Overload ERROR signal
  • PD5: Parallel bus READY signal
  • PD6: Parallel bus ACK signal
